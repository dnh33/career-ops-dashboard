CV Upload & Profile Enrichment

Allow users to upload existing CVs (PDF/DOCX), extract structured data via LLM (Nemotron-3-Ultra), review diffs, and merge into their active profile — enriching the profile context for all future Career Ops LLM interactions.

Architecture Overview

1. Upload
User uploads PDF/DOCX
Frontend: drag-drop or file picker → POST /api/cv/upload-extract (max 10MB)
2. Parse
Text extraction
Backend: pdfplumber (PDF) / python-docx (DOCX) → raw text
3. Extract
LLM structured extraction
Nemotron-3-Ultra via NVIDIA API → Pydantic validated JSON (ExtractedCVData)
4. Review
Diff modal (per-field accept/reject)
Frontend: side-by-side current vs extracted → user curates selection
5. Merge
Persist & enrich
PUT /api/profile (YAML) + PUT /api/cv (MD) + POST /api/profile/enrich (embeddings)
6. Context
Profile context injection
All Career Ops LLM calls include enriched profile in system prompt

# file_parser.py def extract_text_from_pdf(file_bytes: bytes) -> str: with pdfplumber.open(io.BytesIO(file_bytes)) as pdf: return "\n".join(page.extract_text() or "" for page in pdf.pages) def extract_text_from_docx(file_bytes: bytes) -> str: doc = docx.Document(io.BytesIO(file_bytes)) return "\n".join(p.text for p in doc.paragraphs)

LLM Client for Nemotron-3-Ultra

Backend ~45 min

Wrapper around OpenAI-compatible API for NVIDIA's Nemotron-3-Ultra with structured output support.

backend/app/services/llm_client.py backend/app/tests/test_llm_client.py
# llm_client.py class NemotronClient: def __init__(self, api_key: str, base_url: str = "https://integrate.api.nvidia.com/v1"): self.client = openai.AsyncOpenAI(api_key=api_key, base_url=base_url) async def extract_structured(self, prompt: str, schema: dict) -> dict: response = await self.client.chat.completions.create( model="nemotron-3-ultra-256b-v1", messages=[{"role": "user", "content": prompt}], response_format={ "type": "json_schema", "json_schema": {"name": "cv_extraction", "schema": schema} } ) return json.loads(response.choices[0].message.content)
